# MySQL Important Commands Cheat Sheet

## Database Operations
- Show all databases:
  SHOW DATABASES;
- Create a database:
  CREATE DATABASE college;
- Use a database:
  USE college;
- Drop a database:
  DROP DATABASE college;

## Table Operations
- Show all tables:
  SHOW TABLES;
- Create a table:
  CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
  );

## Data Manipulation
- Insert data:
  INSERT INTO students (id, name, age) VALUES (1, 'Amit', 20);
- Select data:
  SELECT * FROM students;
  SELECT name, age FROM students WHERE age > 18;
- Update data:
  UPDATE students SET age = 21 WHERE id = 1;
- Delete data:
  DELETE FROM students WHERE id = 1;


## Table Modification
- Add a column:
  ALTER TABLE students ADD email VARCHAR(100);
- Drop a column:
  ALTER TABLE students DROP COLUMN email;
- Modify a column:
  ALTER TABLE students MODIFY COLUMN name VARCHAR(100);

- Describe table structure:
  DESCRIBE students;
- Drop a table:
  DROP TABLE students;
- Rename a table:
  RENAME TABLE students TO alumni;

## Constraints
- Add primary key:
  ALTER TABLE students ADD PRIMARY KEY (id);
- Add foreign key:
  ALTER TABLE registration ADD FOREIGN KEY (student_id) REFERENCES students(id);

## User Management
- Show users:
  SELECT user, host FROM mysql.user;
- Create user:
  CREATE USER 'himu'@'localhost' IDENTIFIED BY 'mypassword';
- Grant privileges:
  GRANT ALL PRIVILEGES ON college.* TO 'himu'@'localhost';
- Revoke privileges:
  REVOKE ALL PRIVILEGES ON college.* FROM 'himu'@'localhost';
- Drop user:
  DROP USER 'himu'@'localhost';

## Other Useful Commands
- Show current database:
  SELECT DATABASE();
- Show MySQL version:
  SELECT VERSION();
- Exit MySQL:
  EXIT;
